🔤 Strings in Python

Indexing, slicing, and string methods explained simply

1. What is a String?

A string is a sequence of characters enclosed in quotes.

Example

name = "Python"
message = 'Hello World'

2. String Indexing

Each character in a string has an index (position).

Example

word = "Python"

print(word[0])   # P
print(word[1])   # y
print(word[2])   # t

Negative Indexing

print(word[-1])   # n (last character)
print(word[-2])   # o

3. String Slicing

Used to extract a part of a string.

Syntax

string[start:end:step]

Example

word = "Python"

print(word[0:3])   # Pyt
print(word[2:5])   # tho
print(word[:4])    # Pyth
print(word[::2])   # Pto

4. String Methods

Common Methods

text = "python programming"

print(text.upper())      # PYTHON PROGRAMMING
print(text.lower())      # python programming
print(text.title())      # Python Programming
print(text.strip())      # remove spaces
print(text.replace("python", "java"))
print(len(text))         # length of string

5. Useful String Operations

Concatenation

a = "Hello"
b = "World"

print(a + " " + b)

Check substring

text = "Python is easy"

print("Python" in text)   # True

📌 Quick Summary

  • Strings are sequence of characters
  • Indexing starts from 0
  • Negative indexing starts from end
  • Slicing extracts part of string
  • Methods help modify strings
