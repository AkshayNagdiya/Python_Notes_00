🔁 Loops in Python

for loop, while loop, and iteration concepts explained simply

1. What is a Loop?

A loop is used to repeat a block of code multiple times.

2. for Loop

Used to iterate over a sequence (list, string, range).

Example - range()

for i in range(5):
    print(i)

Output

0
1
2
3
4

Example - List

fruits = ["apple", "banana", "mango"]

for f in fruits:
    print(f)

3. while Loop

Runs as long as condition is true.

Example

i = 1

while i <= 5:
    print(i)
    i += 1

Output

1
2
3
4
5

4. break Statement

Used to stop loop immediately.

for i in range(10):
    if i == 5:
        break
    print(i)

5. continue Statement

Skips current iteration.

for i in range(5):
    if i == 2:
        continue
    print(i)

6. Nested Loops

Loop inside another loop.

for i in range(3):
    for j in range(2):
        print(i, j)

📌 Quick Summary

  • for loop → fixed iterations
  • while loop → condition-based
  • break → stop loop
  • continue → skip iteration
  • nested loops → loop inside loop
