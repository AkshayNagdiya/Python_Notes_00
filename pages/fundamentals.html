📘 Fundamentals of Programming

Programming basics, Python setup, and environment understanding

1. What is Programming?

Programming is the process of writing instructions for a computer to perform specific tasks. These instructions are written using a programming language.

Key Points

  • Used to solve real-world problems
  • Works step-by-step (logic based)
  • Requires clear instructions

2. Programming Languages

A programming language is a medium to communicate with computers.

Types

  • High-level (Python, Java)
  • Low-level (C, Assembly)

3. What is Python?

Python is a simple, powerful, and easy-to-learn programming language used in web, AI, automation, and data science.

Why Python?

  • Easy syntax
  • Large library support
  • Fast development

4. Python Setup

Step 1: Install Python

Download from official site:

https://www.python.org

Step 2: Check Installation

python --version

Step 3: Run Python

python filename.py

5. Python Environment

Interpreter Mode

>>> print("Hello Python")

Script Mode

Write code in a file and run using terminal.

print("Hello World")

📌 Quick Summary

  • Programming = writing instructions for computer
  • Python = simple high-level language
  • Setup = install + run via terminal or IDE
  • Two modes = Interpreter & Script
