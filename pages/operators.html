⚙️ Operators in Python

Arithmetic, Comparison, and Logical Operators explained simply

1. Arithmetic Operators

Used to perform mathematical operations.

Operator Description Example
+ Addition 5 + 3 = 8
- Subtraction 5 - 3 = 2
* Multiplication 5 * 3 = 15
/ Division 6 / 2 = 3.0
% Modulus 5 % 2 = 1
// Floor Division 5 // 2 = 2

Example

a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)

2. Comparison Operators

Used to compare two values. Output is True or False.

Operator Description Example
== Equal to 5 == 5 → True
!= Not equal 5 != 3 → True
> Greater than 5 > 3 → True
<< /td> Less than 3 < 5 → True
>= Greater or equal 5 >= 5 → True
<=< /td> Less or equal 3 <= 5 → True

Example

a = 10
b = 5

print(a > b)
print(a == b)

3. Logical Operators

Used to combine multiple conditions.

Operator Description Example
and True if both conditions are true True and True → True
or True if one condition is true True or False → True
not Reverses result not True → False

Example

a = 10
b = 5

print(a > 5 and b < 10)
print(a < 5 or b < 10)
print(not(a > b))

📌 Quick Summary

  • Arithmetic → + - * / % //
  • Comparison → returns True/False
  • Logical → combines conditions
  • Used in decision making and loops
