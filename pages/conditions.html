🔀 Conditional Statements

if, else, elif and match case in Python

1. if Statement

The if statement is used to check a condition. If it is true, the block runs.

Example

age = 18

if age >= 18:
    print("You are eligible to vote")

2. if - else Statement

If condition is false, else block runs.

Example

age = 16

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")

3. if - elif - else

Used when multiple conditions are required.

Example

marks = 75

if marks >= 90:
    print("A Grade")
elif marks >= 75:
    print("B Grade")
elif marks >= 50:
    print("C Grade")
else:
    print("Fail")

4. Nested Conditions

Conditions inside conditions.

Example

age = 20
country = "India"

if age >= 18:
    if country == "India":
        print("Eligible in India")

5. Match Case (Python 3.10+)

Used as an alternative to multiple if-elif conditions.

Example

day = 2

match day:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case _:
        print("Invalid Day")

📌 Quick Summary

  • if → checks condition
  • else → runs if condition is false
  • elif → multiple conditions
  • nested → condition inside condition
  • match case → modern switch alternative
