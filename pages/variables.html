📦 Variables & Data Types

Understanding variables, data storage, and Python data types

1. What is a Variable?

A variable is a container used to store data values in Python. It allows us to reuse and modify data easily.

Example

name = "Python"
age = 20

2. Creating Variables

In Python, you don't need to declare a type. Just assign a value.

x = 10
y = 3.14
z = "Hello"

3. Variable Naming Rules

  • Must start with letter or underscore (_)
  • Cannot start with number
  • No spaces allowed
  • Case-sensitive (name ≠ Name)

4. Data Types in Python

Type Description Example
int Whole numbers 10
float Decimal numbers 3.14
str Text data "Hello"
bool True/False True
list Collection (mutable) [1, 2, 3]
tuple Collection (immutable) (1, 2, 3)
dict Key-value pairs {"name": "Alex"}

5. Collections in Python

List

numbers = [1, 2, 3, 4]

Tuple

numbers = (1, 2, 3)

Dictionary

student = {
  "name": "Akshay",
  "age": 20
}

6. Check Data Type

x = 10
print(type(x))

📌 Quick Summary

  • Variables store data values
  • Python is dynamically typed
  • Common types: int, float, str, list, dict
  • Collections store multiple values
