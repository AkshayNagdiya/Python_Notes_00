⌨️ Input / Output in Python

Learn how to take input, display output, and format results

1. Output using print()

The print() function is used to display output on the screen.

Example

print("Hello World")

Output

Hello World

2. Input using input()

The input() function is used to take input from the user.

Example

name = input("Enter your name: ")
print(name)

Explanation

  • User enters value from keyboard
  • Value is stored as string
  • Then printed using print()

3. Type Conversion in Input

By default, input is always string. We convert it when needed.

Example

age = int(input("Enter age: "))
print(age)

4. Output Formatting

Using f-string (Best Method)

name = "Akshay"
age = 20

print(f"My name is {name} and I am {age} years old")

Output

My name is Akshay and I am 20 years old

5. Multiple Inputs

a, b = input("Enter two numbers: ").split()
print(a)
print(b)

Converted to integers

a, b = map(int, input("Enter numbers: ").split())
print(a + b)

📌 Quick Summary

  • print() → shows output
  • input() → takes input from user
  • input is always string
  • use int() or float() for conversion
  • f-string is best for formatting
