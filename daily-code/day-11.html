1. What is Machine Learning?

Machine Learning is a branch of Artificial Intelligence where computers learn patterns from data and make predictions without being directly programmed.

Example


Input:

Previous house prices


Learning:

Find price patterns


Output:

Predict new house price

2. Artificial Intelligence vs Machine Learning vs Deep Learning

Deep Learning Uses neural networks for complex tasks

3. Types of Machine Learning

  • Supervised Learning
  • Unsupervised Learning
  • Reinforcement Learning

4. Supervised Learning

Supervised learning uses labeled data where the correct output is already known.

Examples

  • Email Spam Detection
  • House Price Prediction
  • Disease Prediction

Types

Type Used For
Regression Predict numerical values
Classification Predict categories

5. Unsupervised Learning

Unsupervised learning finds hidden patterns from data without labels.

Examples

  • Customer Segmentation
  • Grouping Similar Products
  • Market Analysis

Common Algorithms


K-Means Clustering

PCA

Association Rules

6. Reinforcement Learning

Reinforcement learning learns by trial and error using rewards and penalties.

Components

  • Agent
  • Environment
  • Action
  • Reward

Example


Game AI

Agent:

Action:

Move


Reward:

Score
