1. What is NumPy?

NumPy (Numerical Python) is a Python library used for scientific computing and working with numerical data.

Why NumPy?

  • Fast calculations
  • Works with large data
  • Supports mathematical operations
  • Used in Data Science and AI


import numpy as np

2. NumPy Arrays

Array is a collection of elements stored in a single structure.

Creating Array



import numpy as np



numbers = np.array(
[10,20,30,40]
)



print(numbers)
Output: 

[10 20 30 40]

8. Reshaping Arrays

Changing the shape of array without changing data.

[1,2,3,4,5,6]
)

3
)



print(new_array)

[
[1 2 3]
[4 5 6]
