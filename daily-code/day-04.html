1. Lists

A list is a collection that stores multiple values in a single variable.

Creating List


numbers = [10,20,30,40]


print(numbers)
Output: 

[10,20,30,40]

Access List Items


numbers = [10,20,30]


print(numbers[0])
Output: 

10

List Methods


numbers.append(50)

numbers.remove(20)

numbers.sort()

2. Tuples

Tuple is an ordered collection that cannot be changed.



data = (
"Python",
"Java",
"C++"
)



print(data)
Output: 

('Python','Java','C++')

Tuple Access


print(data[0])
Output: 

Python

3. Sets

Set stores unique values and does not allow duplicates.



numbers = {

1,2,3,3,4

}


print(numbers)
Output: 

{1,2,3,4}

Set Methods


add()

remove()

union()

intersection()

4. Dictionaries

Dictionary stores data in key-value pairs.



student = {


"name":"Alex",

"age":20,

"course":"Python"


}



print(student)
Output: 

{
'name':'Alex',
'age':20,
'course':'Python'
}

Access Values



print(student["name"])
Output: 

Alex

5. Comprehensions

Comprehension provides a shorter way to create collections.

List Comprehension



numbers = [

x for x in range(5)

]


print(numbers)
Output: 

[0,1,2,3,4]

Condition Example



even = [

x for x in range(10)

if x % 2 == 0

]


print(even)

6. Nested Data Structures

A data structure inside another data structure is called nested data structure. 



students = [


    {

    "name":"Alex",

    "marks":[80,90,85]

    },


    {

    "name":"John",

    "marks":[70,75,80]

    }


]



print(students)
