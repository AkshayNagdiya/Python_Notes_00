1. What is OOP?

Object Oriented Programming is a programming approach based on objects and classes.

Main Concepts

  • Class
  • Object
  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction

2. Classes and Objects

A class is a blueprint for creating objects. An object is an instance of a class.

Creating Class


class Student:


    name = "Alex"

    age = 20



student1 = Student()



print(student1.name)
Output: 

Alex

3. Constructors

Constructor is a special method that runs automatically when object is created. 



class Student:


    def __init__(self,name,age):

        self.name = name

        self.age = age




student = Student(
"Alex",
20
)



print(student.name)
Output: 

Alex

4. self Keyword

self represents the current object of the class.



class Car:


    def show(self):

        print("This is car")



c = Car()


c.show()

5. Encapsulation

Encapsulation means wrapping data and methods together inside a class. 



class Account:


    def __init__(self,balance):

        self.__balance = balance



    def show_balance(self):

        print(self.__balance)




acc = Account(5000)



acc.show_balance()

6. Inheritance

Inheritance allows one class to use properties of another class.



class Animal:


    def sound(self):

        print("Animal Sound")




class Dog(Animal):

    pass




d = Dog()


d.sound()
Output: 

Animal Sound

7. Polymorphism

Polymorphism means same method name with different behavior.



class Cat:


    def sound(self):

        print("Meow")




class Dog:


    def sound(self):

        print("Bark")




animals = [
Cat(),
Dog()
]



for animal in animals:

    animal.sound()
Output: 

Meow

Bark

8. Abstraction

Abstraction hides implementation details and shows only required information. 



from abc import ABC, abstractmethod



class Shape(ABC):


    @abstractmethod

    def area(self):

        pass




class Circle(Shape):


    def area(self):

        print("Circle Area")

Practical Activities

1. Employee Management System



class Employee:


    def __init__(self,name,salary):

        self.name = name

        self.salary = salary



    def display(self):

        print(
        self.name,
        self.salary
        )




emp = Employee(
"Alex",
50000
)



emp.display()

2. Banking Application



class BankAccount:


    def __init__(self,balance):

        self.balance = balance



    def deposit(self,amount):

        self.balance += amount



    def withdraw(self,amount):

        self.balance -= amount



account = BankAccount(1000)



account.deposit(500)



account.withdraw(200)



print(account.balance)
Output: 

1300
