1. What is File Handling?

File Handling is the process of creating, reading, writing, updating and deleting files using Python.

Files allow data to be stored permanently even after the program stops running.

Why Do We Need File Handling?

Variables store data temporarily in memory. Once the program closes, the data is lost. 


name = "Akshay"

print(name)

After the program closes, the value stored in the variable is removed from memory.

File handling helps us save data permanently so that it can be accessed later even after the program is closed.

Benefits of File Handling

  • Stores Data Permanently
  • Allows Data Sharing Between Programs
  • Maintains Records for Future Use
  • Useful for Large Applications
  • Supports Data Backup and Recovery

Real World Examples

  • User Data Storage
  • Student Management Systems
  • Bank Transaction Records
  • Application Log Files
  • Inventory Management Systems
  • Configuration Files
  • Report Generation Systems

Real Project Example

In a Contact Management System, user information can be stored inside a file instead of keeping it only in variables. 


name = "Akshay"

with open(
"contacts.txt",
"w"
) as file:

    file.write(name)

Now the contact information remains available even after the program is closed.

2. Opening and Closing Files

Before reading or writing a file, we must open it. After completing the operation, the file should be closed.

Python provides the open() function to open files.

Syntax


file = open(
"filename",
"mode"
)

The first argument is the file name and the second argument specifies the file mode.

Opening a File


file = open(
"data.txt",
"r"
)

print(
file.read()
)

file.close()

Why Close a File?

Closing a file releases system resources and ensures that all changes are properly saved. 


file = open(
"notes.txt",
"w"
)

file.write(
"Python"
)

file.close()

What Happens Internally?

  1. Python opens the file.
  2. Data is read or written.
  3. File is closed.
  4. Resources are released.

Real World Example

Opening a file is similar to opening a notebook. You open it, read or write information, and then close it.

Interview Question

The open() function is used to open a file in Python. The close() method is used to close the file and release system resources.

3. File Modes

File modes determine how a file will be used after opening. Different modes allow reading, writing, appending or creating files.

Common File Modes

Mode Description
r Read File (Default Mode)
w Write File (Creates New or Overwrites Existing File)
a Append Data to Existing File
x Create New File Only
rb Read Binary Files
wb Write Binary Files

Read Mode (r)

Used to read data from an existing file. 


file = open(
"data.txt",
"r"
)

print(
file.read()
)

file.close()

Write Mode (w)

Used to write data into a file. Existing data will be removed. 


file = open(
"notes.txt",
"w"
)

file.write(
"Python Learning"
)

file.close()

Append Mode (a)

Adds new content without deleting existing data. 


file = open(
"notes.txt",
"a"
)

file.write(
"\nNew Note"
)

file.close()

Create Mode (x)

Creates a new file. Generates an error if the file already exists. 


file = open(
"newfile.txt",
"x"
)

file.close()

Binary Modes

Binary modes are used for images, videos, PDFs and other non-text files. 


file = open(
"image.jpg",
"rb"
)

data = file.read()

file.close()

When to Use Which Mode?

Situation Mode
Read Existing Data r
Create New File w or x
Add New Data a
Work with Images/PDFs rb / wb

4. Reading Files

Reading a file means retrieving data stored inside a file.

Python provides several methods to read file contents.

Reading Entire File


file = open(
"data.txt",
"r"
)

content = file.read()

print(content)

file.close()

Output: 


Hello Python

Welcome to File Handling

read() Method

The read() method reads the entire content of a file. 


file = open(
"data.txt",
"r"
)

print(
file.read()
)

file.close()

Reading Specific Characters


file = open(
"data.txt",
"r"
)

print(
file.read(5)
)

file.close()

Output:


Hello

readline() Method

Reads only one line at a time. 


file = open(
"data.txt",
"r"
)

print(
file.readline()
)

file.close()

Output:


Hello Python

readlines() Method

Reads all lines and returns them as a list. 


file = open(
"data.txt",
"r"
)

lines = file.readlines()

print(lines)

file.close()

Output:


['Hello Python\n', 'Welcome to File Handling']

Reading File Using Loop


file = open(
"data.txt",
"r"
)

for line in file:

    print(line)

file.close()

This is useful when working with large files because it reads one line at a time.

Real World Example

Reading user data from a file. 


file = open(
"users.txt",
"r"
)

for user in file:

    print(user)

file.close()

When to Use Which Method?

Method Use Case
read() Read Entire File
read(size) Read Limited Characters
readline() Read One Line
readlines() Read All Lines as List

5. Writing Files

Writing a file means storing data inside a file.

Python uses the write() method to add content to a file.

Why Do We Need Writing Files?

Writing files allows us to save data permanently instead of storing it only in variables.

Basic Example


file = open(
"notes.txt",
"w"
)

file.write(
"Python File Handling"
)

file.close()

The content is saved inside notes.txt.

Output in File


Python File Handling

Writing Multiple Lines


file = open(
"notes.txt",
"w"
)

file.write(
"Python\n"
)

file.write(
"Java\n"
)

file.write(
"JavaScript"
)

file.close()

Output in File:


Python

Java

JavaScript

Using writelines()

writelines() is used to write multiple lines at once. 


languages = [

"Python\n",

"Java\n",

"C++\n"

]

file = open(
"languages.txt",
"w"
)

file.writelines(
languages
)

file.close()

Important Note

Opening a file in write mode (w) removes all existing content before writing new data. 


file = open(
"notes.txt",
"w"
)

file.write(
"New Content"
)

file.close()

Previous data will be deleted.

Real World Example

Saving user information into a file. 


name = "Akshay"

file = open(
"user.txt",
"w"
)

file.write(name)

file.close()

Common Writing Methods

Method Purpose
write() Write Single String
writelines() Write Multiple Strings

5. CSV File Handling

CSV stores data in table format.



import csv



with open(
"students.csv",
"w"
) as file:


    writer = csv.writer(file)



    writer.writerow(
    [
    "Name",
    "Age"
    ]
    )

Reading CSV



import csv



with open(
"students.csv"
) as file:


    data = csv.reader(file)



    for row in data:

        print(row)
