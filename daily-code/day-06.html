1. File Handling

File handling is used to create, read, update and delete files.

Opening File


file = open(
"data.txt",
"r"
)


print(file.read())


file.close()

2. File Modes

Mode Purpose
r Read file
w Write file
a Append data
x Create file

3. Reading and Writing Files

Write File



file = open(
"notes.txt",
"w"
)



file.write(
"Python File Handling"
)



file.close()

Read File



file = open(
"notes.txt",
"r"
)



data = file.read()



print(data)



file.close()
Output: 

Python File Handling

4. Using with Statement

with automatically closes the file after operation.



with open(
"data.txt",
"r"
) as file:


    print(
    file.read()
    )

5. CSV File Handling

CSV stores data in table format.



import csv



with open(
"students.csv",
"w"
) as file:


    writer = csv.writer(file)



    writer.writerow(
    [
    "Name",
    "Age"
    ]
    )

Reading CSV



import csv



with open(
"students.csv"
) as file:


    data = csv.reader(file)



    for row in data:

        print(row)

6. JSON File Handling

JSON is commonly used for storing and transferring data.



import json



student = {


"name":"Alex",

"age":20

}



with open(
"student.json",
"w"
) as file:


    json.dump(
    student,
    file
    )

Reading JSON



with open(
"student.json",
"r"
) as file:


    data = json.load(file)



print(data)

7. Exception Handling

Exception handling prevents program crashes.



try:


    number = int(
    input("Enter number: ")
    )


    print(number)



except:


    print(
    "Invalid Input"
    )

8. try except finally



try:


    x = 10 / 2



except ZeroDivisionError:


    print(
    "Cannot divide"
    )



finally:


    print(
    "Program Finished"
    )

9. Custom Exceptions

Creating your own error using Exception class.



class AgeError(Exception):

    pass




age = int(
input("Age: ")
)



if age < 18:


    raise AgeError(
    "Not Allowed"
    )
