1. What is Pandas?

Pandas is a Python library used for data analysis and data manipulation.

Why Pandas?

  • Handle large datasets
  • Clean data easily
  • Analyze information
  • Prepare data for Machine Learning


import pandas as pd

2. Pandas Series

Series is a one dimensional labeled data structure.



import pandas as pd



numbers = pd.Series(
[10,20,30,40]
)



print(numbers)
Output: 

0    10
1    20
2    30
3    40

3. DataFrame

DataFrame is a two dimensional table structure.



data = {


"Name":[
"Alex",
"John",
"Mike"
],


"Age":[
20,
21,
22
]


}



df = pd.DataFrame(data)



print(df)
Output: 

Name   Age

Alex   20

John   21

Mike   22

df.head()



df.tail()

6. Data Cleaning

Data cleaning means fixing incorrect or incomplete data.

Check Missing Values



df.isnull()

Remove Missing Data



df.dropna()

Fill Missing Data



df.fillna(0)

7. Filtering Data



students = df[
df["Age"] > 20
]



print(students)
Output: 

Students having age greater than 20

8. Sorting Data



df.sort_values(
"Age"
)

Sorts data according to column values.

9. Group By Operations

GroupBy is used to group data and perform calculations.



data = {


"Department":[
"IT",
"IT",
"HR"
],


"Salary":[
50000,
60000,
40000
]


}



df = pd.DataFrame(data)



result = df.groupby(
"Department"
).sum()



print(result)
