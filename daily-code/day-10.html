1. What is Data Visualization?

Data visualization is the process of representing data using charts and graphs to understand patterns easily.

Why Visualization?

  • Find hidden patterns
  • Compare values
  • Understand trends
  • Make better decisions

2. Visualization Libraries

Library Use
Matplotlib Basic and customizable charts
Seaborn Advanced statistical visualization 


import matplotlib.pyplot as plt

import seaborn as sns

3. Line Chart

Line chart shows changes or trends over time.



import matplotlib.pyplot as plt



months = [
"Jan",
"Feb",
"Mar",
"Apr"
]


sales = [
100,
200,
150,
300
]



plt.plot(
months,
sales
)



plt.title(
"Monthly Sales"
)


plt.xlabel(
"Month"
)


plt.ylabel(
"Sales"
)


plt.show()

4. Bar Chart

Bar chart compares different categories.



import matplotlib.pyplot as plt



products = [
"Mobile",
"Laptop",
"Tablet"
]


price = [
500,
1000,
700
]



plt.bar(
products,
price
)



plt.title(
"Product Price"
)



plt.show()

5. Pie Chart

Pie chart represents percentage distribution.



import matplotlib.pyplot as plt



values = [
40,
30,
20,
10
]


labels = [
"Python",
"Java",
"C++",
"JS"
]



plt.pie(
values,
labels=labels,
autopct="%1.1f%%"
)



plt.show()

6. Histogram

Histogram shows frequency distribution of data.



import matplotlib.pyplot as plt



marks = [
40,
50,
60,
70,
80,
90
]



plt.hist(
marks
)



plt.show()

7. Scatter Plot

Scatter plot shows relationship between two variables.



import matplotlib.pyplot as plt



height = [
150,
160,
170,
180
]


weight = [
50,
60,
70,
80
]



plt.scatter(
height,
weight
)



plt.show()

8. Seaborn Visualization

Seaborn provides beautiful statistical charts.



import seaborn as sns

import matplotlib.pyplot as plt



data = sns.load_dataset(
"iris"
)



sns.scatterplot(
x="sepal_length",
y="petal_length",
data=data
)



plt.show()

9. Heatmap

Heatmap displays correlation using colors.



import seaborn as sns



sns.heatmap(
data.corr(),
annot=True
)

10. Box Plot

Box plot helps find spread and outliers.



sns.boxplot(
data=data
)

11. Distribution Plot



sns.histplot(
data["sepal_length"],
kde=True
)

Shows data distribution with density curve.
