1. Conditional Statements

Conditional statements are used to make decisions in a program.

if Statement


age = 20


if age >= 18:

    print("You can vote")

Output


You can vote

2. if else Statement

if else executes different blocks based on condition.


number = 10


if number % 2 == 0:

    print("Even Number")


else:

    print("Odd Number")

Output


Even Number

3. if elif else Statement

Used when multiple conditions are required.



marks = 75



if marks >= 90:

    print("Grade A")


elif marks >= 60:

    print("Grade B")


else:

    print("Fail")

Output


Grade B

4. Nested Conditions

A condition inside another condition is called nested condition.



age = 20


if age >= 18:


    if age <= 60:

        print("Eligible")
Output: 

Eligible

5. Loops

Loops are used to repeat a block of code multiple times.

For Loop



for i in range(5):

    print(i)
Output: 

0
1
2
3
4

6. Range Function



range(start, stop, step)
Example: 


for i in range(1,6):

    print(i)
Output: 

1
2
3
4
5

7. Loop Control Statements

break



for i in range(10):

    if i == 5:

        break

    print(i)

continue



for i in range(5):

    if i == 2:

        continue

    print(i)
