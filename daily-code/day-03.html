Creating Reusable Code Using Functions

1. What is Function?

A function is a reusable block of code that performs a specific task.

Why use Functions?

  • Avoid repeating code
  • Make code clean
  • Easy debugging
  • Better project structure

2. Creating a Function

Functions are created using the def keyword.


def greet():

    print("Hello Python")


greet()

Output


Hello Python

3. Parameters and Arguments

Parameters are values received by a function. Arguments are values passed when calling a function. 



def welcome(name):

    print(
    "Welcome",
    name
    )



welcome("Alex")

Output


Welcome Alex

4. Multiple Parameters



def add(a,b):

    result = a + b

    print(result)



add(10,20)
Output: 

30

5. Return Values

return sends a value back from a function.



def square(number):

    return number * number



result = square(5)



print(result)
Output: 

25

6. Variable Scope

Local Variable



def test():

    x = 10

    print(x)



test()

Variable created inside function is local.

Global Variable



x = 100



def show():

    print(x)



show()

2. Factorial Calculator



def factorial(n):

    result = 1


    for i in range(1,n+1):

        result *= i


    return result




num = int(
input("Enter number: ")
)



print(
factorial(num)
)
Output: 

120

3. Menu Driven Program



def add(a,b):

    return a+b



def subtract(a,b):

    return a-b



print("1.Add")

print("2.Subtract")

)



a = 10

    print(
